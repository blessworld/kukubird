<br>About kukubird
<br>----------------
<br>python + webpy + jquery mobile搭建的看漫画小站。 目前运行在sae上。 http://kukubird.sinaapp.com
<br>这个是学习webapp开发的衍生作品，真正用心做了。
<br>目前在Firefox、Chrome、opera中都测试没问题。针对IOS做了特别优化，在IPAD上体验很不错。在Android平板上，用opera、QQ浏览器、Firefox等可以凑合看。
<br>
<br>后端是独立的，目前实现了:
<br>后端添加漫画地址，服务器脚本自动下载，前端的漫画列表会自动更新。
<br>
<br>实现思路可以参见以下我的blog:
<br>http://my.oschina.net/memorybox/blog/55349
<br>
<br>文档目录:
<br>jquerymobile文件夹为客户端，均为静态html，js脚本。
<br>其余为服务端及工具脚本。
